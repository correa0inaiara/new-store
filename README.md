## About
This is an E-commerce Department Store built for my portfolio.

## Status
Currently in development - Features are being added incrementally

## Features
- **User Authentication**: Secure login and registration with Clerk

## Tech Stack

### Frontend
- **Next.js 16**: Full-stack React Framework with TypeScript
- **Tailwind CSS**: CSS Framework
- **DaisyUI**: Component library for Tailwind CSS

### Backend
- **Prisma**: TypeScript ORM for database management
- **PostgreSQL**: Relational Database for data persistence

### Authentication & Services
- **Clerk**: Complete user management and authentication
